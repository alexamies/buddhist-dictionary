折疑論卷第一
金臺大慈恩寺西域師子比丘述註

[0794c23]  (曲而斷之謂折。猶豫不決之謂疑。評議難辨之謂論。此論因妙明子居山時有客特詣請問以決所疑。
妙明子引三教微言以答之。遂成是錄。故曰折疑)  。

敘問第一  (敘者。敘其來因。問者問其所疑。第者次第也。一者數之始也)

[0794c26]  妙明子  (華州渭南縣洪福寺出家。遇魚巖老人祥公處得法。闡揚大乘經律論。大元至正間。詔封  傳大乘戒賜紫閏大國師)  。

[0794c28]  居石室  (後歸終南山石室隱居)  。

[0794c29]  有客來謁。妙明與之坐而問焉曰。夫子何來  (尊美之稱曰夫子。謂客因何事而來)  。

[0795a02]  客曰。有所請問而來  (客曰。有所疑而來問)  。

[0795a03]  妙明曰。吾居是山也。幾以數年而世罕知  (吾者  我也。幾者。近也。罕者少也謂我居此山。將近十數年矣。而世人少知)  。

[0795a05]  加以懸石巉巖危嶠巇嶮。無人履踐  (巉音讒。險也。巖積石  貌。嶠渠廟切。山銳而高也  巇
音羲。嶺危也。嶮音。險音險也。加以者兼有也。懸石者纍垂可畏貌。謂險難行  之處少有往來。故曰無人履踐也)  。

[0795a08]  遠勞仁者一至此乎  (遠路勞煩仁者一徑來於此乎)  。

[0795a09]  客曰。雷霆之聲無欲聞人之耳。而耳自聞  (淮南  子曰。雷者陰陽擊摶之聲。感而為雷。激而為霆。雷聲豈要聞於人耳蓋人耳自皆聞之)  。

[0795a11]  管籥之音無有悅人之心。而心自悅  (樂譜云。管者簫笛之  屬。籥者似笛而有三孔短小。其管籥音聲無要歡悅於人心。蓋人聞之則自然歡悅也)
。

[0795a13]  夷齊匿跡於首陽。不欲德稱於人。而人自稱  其德
(史記。伯夷叔齊孤竹君之二子。讓不嗣位欲之西南。遇武王伐紂。迎馬首而諫不從。隱於首陽山。  誓不食周粟。卒至餓死矣)  。

[0795a16]  巢許韜光於箕岫。豈為道揚於世。而世自揚  其道  (巢地名韜藏也。莊子讓王篇曰。堯以天下讓許由。許由不受。隱於箕岫山)  。

[0795a18]  子之居山亦以類矣。奚為人之不知也  (客以夷齊許由  之德而比類。何為人之不知也)  。

[0795a20]  妙明曰。子所欲問可得聞乎  (此審問辭)  。

[0795a21]  客曰。請無拒焉  (無禁止之辭。子無阻而抵拒焉)  。

[0795a22]  妙明曰。子張云。君子尊賢而容眾。嘉善而  矜不能。我之大賢歟。於人無所不容。我之不  賢歟。人將拒我。如之何其拒人也
(論語。子夏門人問交於  子張。子張曰。子夏云何。對曰子夏曰。可者與之。不可者拒之。子張曰。異乎吾所聞。君子尊賢而容眾云云。蓋子
夏之言迫狹。子張譏之是也。子張所言容字正破子夏拒字)  。

[0795a27]  知則無拒  (此許答之辭也)  。

[0795b01]  客曰噫  (噫者傷痛悲歎之辭)  。

[0795b02]  吾與子皆儒也  (我與子皆曾為儒也)  。

[0795b03]  予昔嘗與子為方外友  (予我也。昔者往日也。嘗曾也。撤去籓籬出乎流俗之  表曰方外也。同志曰友)  。

[0795b05]  獨以兵火之亂。不得同地而處。良可歎也  (獨者  單也。言單為兵亂而逃避分散也。逃避不得同居。良頗亦可悲歎也)  。

[0795b07]  子下髮離俗。其有年矣  (子落髮為僧離去俗家。有多年矣)  。

[0795b08]  嘗欲一往而見。不果如願  (曾要專一來此見面。不如願欲)  。

[0795b09]  頃以稅駕塵習撥置人事。姑乘少暇聿來高  居。欲折所疑。莫知可否  (否音府。否者審辭也。頃者俄頃少時也。以用也。稅駕
者農夫之具。頃以脫鞅憩歇暫放下之義。塵習者塵寰家務之習。撥置者謂棄之於一邊。人事者往來交接之事也。
姑且因乘少時閑暇聿遂而來高隱之居欲以決折所疑之惑情。莫知可許否)  。

[0795b13]  妙明曰。請試言之  (請試言所疑之辭)  。

[0795b14]  客曰。人云  (假借他人之言)  。

[0795b15]  曩妙明儒時。未嘗寄口於佛經  (曩昔也。言往昔之日為儒。時未  嘗掛口於佛經)  。

[0795b17]  今為釋也。沈湎因果。謂佛教也空寂之說  (今為  釋子如沈貪酒色心迷意醉亦如湎波溺而不醒。乃妄為因緣果報虛無空寂之論說)  。

[0795b19]  遺棄周孔黃老之書。不識有諸  (遺猶棄也。謂遺棄周公孔子黃帝老  聃之經書)  。

[0795b21]  妙明曰。子貢云。君子一言以為知。一言以  為不知。言不可慎也  (知皆音智。論語。陳子禽謂子貢曰。子為恭也。仲尼豈賢於
子乎。言謂恭者謂恭敬。推遜其師也。子貢曰。君子一言以為知云云。此責子禽不謹言也。蓋一言善為智。一言不  善為不智。智不知係於一言。不可不謹慎也)  。

[0795b25]  凡物自微至著緣淺及深  (微者。細也小也。著者盛也。大也。凡一切物自細  小而至盛大。緣者由也。譬人入海由淺而以及其深也)  。

[0795b27]  譬登萬仞之嶽不可一步而至。必資下以及  其高
(顏師古書曰。七尺為仞。嶽者山之高廣者也譬喻人登萬仞之山嶽不可一步便到。必自下以至其高上)  。

[0795c02]  既登絕頂。所歷諸山。豈曰棄哉  (既登絕高之頂。觀經歷之眾山  皆低未曾遺棄也)  。

[0795c04]  吾始學周孔黃老之書。抑以為極致  (我始初為學周孔等書之  時。抑以為至絕之處)  。

[0795c06]  次撿佛經。始知聖道有權實爾  (次因檢閱佛之經典。始知聖道有  權假真實之理故爾)  。

[0795c08]  今吾以姬孔黃老之書為眾山  (姬周文王姓。謂周孔黃老四聖之書。  喻如眾山)  。

[0795c10]  以佛經第一義諦為絕頂  (第一義諦者。教乘之極談也。已達聖位者為真諦。
未脫凡情者為俗諦。於時真俗不立凡聖情盡。不著二邊唯存中道。故曰第一義諦。以此喻絕高之頂)  。

[0795c12]  既登絕頂。望眾山自遠矣。何棄之有耶  (既明釋典  如登絕高之頂。復觀周孔等書如眾山而自遠矣。何曾遺棄之有耶也)  。

聖生第二  (此篇言佛降誕之跡。故曰聖生)

[0795c15]  客曰。佛之所生在何邦域。其氏族顯跡及形  貌德相所說法要可得聞乎  (佛之降生國土姓氏家族兼及形相所談妙  法。亦可得而聞乎)  。

[0795c18]  妙明曰。富哉問也  (發問之辭深遠廣大故曰富問也)  。

[0795c19]  予雖不敏。略示所知以報吾子  (我雖不敏達略將少知用答子之  問也)  。

[0795c21]  自漢之西。過數萬里  (自漢者指中華漢地也。之者往也。自漢地往西約有數萬里程  途也)  。

[0795c23]  有國曰迦維羅  (被有國土名迦維羅乃佛降誕之中國也)  。

[0795c24]  王姓釋迦  (釋氏源流梵語釋迦。此云能仁)  。

[0795c25]  父名淨飯  (梵語首頭陀羅。或云悅頭檀。此云淨飯)  。

[0795c26]  大夫人曰摩耶  (梵語摩耶。此云大術)  。

[0795c27]  佛本居兜率天宮  (梵語兜率陀。亦名覩一史多。此云知足也)  。

[0795c28]  以先佛之迦葉般涅槃後佛當補處。名曰護  明
(梵語具云摩訶般涅槃那此云大滅度。亦云大寂滅。自迦葉佛涅槃之後。佛補其一處。特待出現之時)  。

[0796a02]  實以三祇因滿  (一切諸佛皆修三祇之因也。三祇者。即三大阿僧祇劫也)  。

[0796a03]  萬德果圓  (諸佛皆具萬德方為道果圓滿也)  。

[0796a04]  宿願未周  (宿生有願。未曾同圓。故下生人間)  。

[0796a05]  示同生滅  (示現受生滅身)  。

[0796a06]  駕日輪香象。託陰王宮  (初投胎時。化現於日輪之中跨香象。投託母胎於王宮)  。

[0796a07]  以大夫人摩耶為母。以周昭王甲寅二十四  年四月八日  (當周之時正月建子。即今二月八日是也)  。

[0796a09]  毘藍園中右脇而生  (脅生者表異常流。周書異記云。昭王二十四年四月八日。此方
江河泛漲大地皆動。光貫太微照自西方。王即怪問群臣。太史蘇由奏曰。西方生大聖人也。王曰。於此何如。由曰
無事。千年之後聲教傳流。於此時乃刻銘於國之南郊以記之)  。

[0796a12]  於時地搖六震  (六震。即六種震動。謂動踊震起吼擊也。搖揚不安曰動。  [嶙-山+土]  瓏凹凸曰踊。
隱隱有聲曰震。自下昇高曰起。砰磕發響曰吼。打摶驚物曰擊。然各有三。直動名動四天下動名遍動。盡大千動名  等遍動。餘五例此)  。

[0796a15]  天雨四華  (雨音裕。降也。四華者。曼殊沙華。摩訶曼殊沙華。芬陀梨華。拘物頭華)  。

[0796a16]  神捧金盤  (地神捧持金盤)  。

[0796a17]  龍吐香露  (九龍吐其香水浴太子身)  。

[0796a18]  即能縱行七步  (周迴行七步。七步蓮華皆生。七步者。表七種淨財。亦名七德財。即信進戒  慚愧聞捨定慧七也)  。

[0796a20]  目顧四方  (眼顧視於東西南北四方。本行經云。東方表涅槃最上。南方表利樂眾生西方表解脫。北  方表永斷輪迴)  。

[0796a22]  一手指天一手指地。曰天上天下唯吾獨尊  (目顧  四方。上下所指。一則異於凡流。二者發明大事因緣。翫者於此動用之間。自可消息)  。

[0796a24]  具三十二大人相  (阿毘曇論云。三十二相者。梵音聲相眉間白毫相。烏瑟尼妙相。牛王
睫相。目紺青相。廣長舌相。得上味相。齒牙鮮白相。齒齊。互密相。具四十二齒相。師子頷輪相。立身摩膝相。
肩膊圓滿相。師子身相。廣洪直相處處充滿相。皮膚細軟相。常光一尋相身真金色相。身毛右旋相。孔生一毛
相。身毛上靡相。身分圓滿相。勢峯藏密相。伊尼鹿王腨相。足趺滿厚相手指網縵相。手足細軟相。足踉圓滿相。  纖長光澤相。足千輻輪相。足底平滿相)  。

[0796b04]  八十種隨形好  (八十種好者。指甲紅赤。指甲隆起。指甲清淨指甲滿足。押甲團圓指甲纖
直。指間一密。指清淨潔。手足肥膩。手足裏赤。手足平等。手足內滿。掌文深現。掌文端直。掌文纖長。手足潤
澤。掌文不亂。踝足少現。膝頭圓滿。膝次第滿。行步齊整師子王行。鵝王步。龍王行相。牛王行相。行不顧視。行步
不亂。半身正直。佛身過人。一切滿足。佛身皆好。身體平正。身體遍足。身體正直。身體滑澤。身體淨潔。身體端
嚴。身次第大小。諸根方正。身色不黑。身體無黶。身毛淨潔。腹相團圓腹無橫文。身體明淨。臍正圓深。臍孔團圓。
臍文右旋。臍孔不凹。臍孔不長。臍孔不短。臍孔不連。得龍牙相。牙不過脣。四牙團圓四牙鋒利。四牙齊密。舌長
柔軟。舌色赤好。梵聲深好。象王聲相。迦陵頻伽聲。齒根圓滿。鼻孔淨潔。鼻不下垂。鼻高脩長。鼻高方廣。目廣清
淨。眼睛黑光。目睫次第眉如半月。眉毛黑澤。眉毛光明。耳皆垂埵頭髮細軟)  。

[0796b12]  王請僊人阿私陀瞻之  (西方能相吉凶貴賤仙人也)  。

[0796b13]  曰。太子隨身相好粲若芬華  (太子身相。如初開芬陀梨華。嬌態香潔也)  。

[0796b14]  在家作轉輪王。出家成無上道  (在家定作金轉輪王出家必成無上  道果)  。

[0796b16]  以臣觀之。必不得在家矣  (仙人所言。太子定離國位)  。

[0796b17]  遂泣曰。太子成佛必施大法。臣今老矣。失。

[0796b18]  此大利。良可悲哉  (有淚無聲曰泣。仙人知太子成佛演說大法老不得聞。故乃泣之)  。

[0796b19]  年十有九。壬申二月八日  (周時正月建子。二月八日即今十二月八日是也)  。

[0796b20]  呼車匿鞁朱騣  (車匿者。佛之家所生之子。朱騣者馬也)  。

[0796b21]  遊看四門。逢僧了法。不樂王宮。亟欲修心  (因遊
觀四門。東門見老人衰朽。南門見其病者。西門見送葬者。北門見一比丘。乃是作瓶天子所化。太子問言汝何等
人。比丘答曰。世間無常。我由修其聖道超於彼岸。若能如是修者。亦皆如之。言訖騰空而去。太子因見老病死苦
甚為恐怖。遇此比丘愈起善心。厭棄王宮。亟急欲要出家修行)  。

[0796b25]  王生憂懼恐斷國嗣。遂增以侍御音樂娛之。  太子不悅  (王知太子心欲出家。恐懼斷絕國之後嗣。遂增添侍奉婇女。加以管絃音樂取其歡樂太
子不以為悅樂也)  。

[0796c03]  迨夜之央  (迨至也。央半也。至其半夜)  。

[0796c04]  淨居天人  (浮居天人。乃是輔太子出家者也)  。

[0796c05]  以天福力。彰侍御之魗容。九孔流溢不淨。蛆  蟲可厭
(魗音醜。淨居天人用天福神力彰顯侍御。綏女俱各昏睡。現死屍相。各兩兩耳兩鼻口并大小  便等。九孔流出不淨蛆蟲。太子轉生厭離之心也)  。

[0796c08]  夜半子時。淨居天人報曰。太子可出宮。今正  是時  (天人告太子曰。既降兜率欲滿宿願。要出王宮今正是時)  。

[0796c10]  於是  [馬*夸]  犍陟  (犍陟者馬名也)  。

[0796c11]  僕車匿  (車匿從人)  。

[0796c12]  天人捧駕  (天人捧其馬足)  。

[0796c13]  飛空而去  (飛空逾城而去)  。

[0796c14]  投身雪嶺。趺坐盤石  (既至雪嶺。坐於盤陀石上)  。

[0796c15]  淨思六年。遂成佛果  (正宗記云。先於彌樓山。習不用處定。次於欝頭弗處。習非
非想處定。又於象鼻山。覩明星面悟道。六年淨思方成佛果矣)  。

[0796c17]  無惡不去。無善不圓  (去上聲。除也。諸惡盡去。萬善周圓)  。

[0796c18]  首詣鹿林三轉四諦。而摩訶男等五人得道  (五人
者。佛初入山。父族三人一阿濕婆。二婆提三摩訶男拘利。母族二人。一憍陳如。二十力迦葉。侍佛佛不用。五人
各修梵行。佛既成道。首於鹿苑。與憍陳如等說三轉四諦法輪。三轉者。一示相轉。二勸修轉。三作證轉。一示相轉
者。此是苦。此是集。此是滅。此是道。二勸修轉者。此是苦汝須知。此是集汝須斷。此是滅汝須證。此是道汝須
修。知世間苦因苦果。出世間樂因樂果也。三作證轉者。此是苦我已知。此是集我已斷。此是滅我已證。此是道我
已修。陳如等疑問。如來還出世間苦因苦果樂因樂果否。如來答云。我已知我已斷。我已修。我已證。五人聞說此  法各證道果)  。

[0796c26]  次演空宗。由是向室羅筏城祇園等處。八部  般若次第興焉  (室羅筏城。即舍衛國城也。祗園者。即祗樹給孤獨也。八部般若者。大品般

[0799a22]  而無不為  (法身常寂。為度生故亦若有為)  。

[0799a23]  有情無情  (有情者。一切含靈也。無情者。闡提也。然闡提有情。而不明真理若無情也)  。

[0799a24]  平等共有  (結上句義。金剛經云。是法平等也)  。

[0799a25]  以如是等無量行相  (用如上所言。喻無量行相)  。

[0799b01]  而非行相  (雖用無量行相以喻之。實不著於行相也)  。

[0799b02]  應正了知名法身佛  (總結上文意。當如上真正了知。方纔名為法身佛也)  。

[0799b03]  客曰。法身之義盡在此乎  (法身義理。盡在此言乎)  。

[0799b04]  妙明曰。抑未也  (抑然而未盡也)  。

[0799b05]  夫法身者  (此下復明法身之妙)  。

[0799b06]  離四句絕百非  (四句者。有句。無句。亦有句。亦無句。非有句非無句也。百非者。四四相資
一十六。三世共有四十八已起未起九十六四種根本百法。謂但著一法即非也。故曰百非)  。

[0799b08]  動念則隔  (動念則情生。情生則智隔)  。

[0799b09]  強言則乖  (強上聲強言則不合理但著言說則乖離也)  。

[0799b10]  脫聲塵  (脫離耳界聲塵)  。

[0799b11]  離文字  (離去語言文字)  。

[0799b12]  而語默動靜。皆不可得而親也  (有言曰語。無言曰默。體搖曰動。  心寂曰靜於此語默動靜之間。皆不可得而親近也)  。

[0799b14]  蓋以法身離言  (總結上文意。大蓋法身離言語。如云不著見聞覺知故也)  。

[0799b15]  離言亦非法身也  (若離語言之外別求法身。亦非法身。如云不離見聞覺知是也)  。

[0799b16]  子頗欲學達離不離言之道者乎  (汝頗少學此離與不離言之妙  道者乎)  。

[0799b18]  客曰。唯  (唯音尾速應之辭也)  。

[0799b19]  妙明曰。欲善其事先利其器  (論語。子貢曰。工欲善其事。必先利其  器。欲為仁而親仁賢欲學聖道。必先立其所志。如此。則聖道易明矣)
。

[0799b21]  子能先學離不離言之方便乎  (善巧易為曰方便也)  。

[0799b22]  客曰。唯命是聽  (速而應曰聽其命令)  。

[0799b23]  妙明曰。欲達離不離言之道者  (此下形容造道之妙)  。

[0799b24]  當先觀一切法  (欲學此道。應當觀察世間所有生滅一切等法)  。

[0799b25]  無實無虛  (莫於虛實而生取著)  。

[0799b26]  無我無彼  (莫生人我之見)  。

[0799c01]  無垢無淨  (本性寂然元無垢淨)  。

[0799c02]  黜聰明  (去智曰黜。謂去世間聰明)  。

[0799c03]  墮支體  (離形曰墮。謂離四大支體)  。

[0799c04]  廓塵習  (廓落塵寰所習)  。

[0799c05]  屏念慮  (屏音秉。謂屏除妄念思慮)  。

[0799c06]  劈情卷慾  (劈破私情卷除嗜慾)  。

[0799c07]  泯智凝神  (泯絕世智凝結心神)  。

[0799c08]  忘懷澄思  (不記情懷澄清思慮。如上數事去之可以近道矣)  。

[0799c09]  然後眼如耳。耳如鼻。鼻如口。口如耳  (凡學道者。到此
心凝形釋之時。故六根總歸一源。身心唯一不二。既爾則色聲香味觸法六塵。皆不能搖)  。

[0799c11]  身心混融。體智圓通  (結上文意。既六根不染。身心混融。圓通一體)  。

[0799c12]  直得貫疎親  (不知親己亦不疎物)  。

[0799c13]  忘物我  (至道不可得。以情求其物我之情。皆不可不忘也)  。

[0799c14]  泯榮辱  (安榮恥辱之情。而盡泯滅)  。

[0799c15]  絕是非  (是與不是。盡皆絕之)  。

[0799c16]  收視返聽  (收  斂  其目返照本源)  。

[0799c17]  虛已冥真  (空虛我有冥契本真)  。

[0799c18]  百醭出於口邊  (忘其語言)  。

[0799c19]  青草生於舌上  (甚如死人)  。

[0799c20]  千聖喚不回頭  (靈心絕待)  。

[0799c21]  百鳥都無尋處  (泯相澄神)  。

[0799c22]  形同槁木  (形體同枯槁之木者。無生意也)  。

[0799c23]  心若死灰  (心若久冷灰者。再不起念也)  。

[0799c24]  智閑遠水孤峯  (莊子云。大智閑。閑者是從容自得之貌。可謂遠水孤峯者。雖有形而取之  不可及也)  。

[0800a01]  性寂寒潭皎月  (性體寂然。若寒潭水中之月影雖動而不搖流而不去矣)  。

[0800a02]  然後可以盡離不離言之道者也  (總結上文意。若能如是。然  後可許為離不離言之道)  。

[0800a04]  客蹶然而興曰  (蹶然速起之貌。興者興起之辭)  。

[0800a05]  予心意懵  [懵-目+(歹*巳)]  (懵毛孔反。悶亂而性暗也。  [懵-目+(歹*巳)]  莫公反。心迷而不明也)  。

[0800a06]  精神慌  [怡-台+莽]  (慌胡廣反。忽也  [怡-台+莽]  音母。愚昧也)  。

[0800a07]  於子之說。莫知其所入也  (總結上文意。悶亂而性暗。心迷而不明。恍惚愚  昧。於子之說法身妙理。皆無所知所入也)  。

[0800a09]  妙明曰。夫何忽哉  (何恍惚之有哉)  。

[0800a10]  始吾以子為達矣。今子之鄙而至此乎  (始初見子為通  達者。今逼急而無所知者。豈不鄙陋而至於此乎)  。

[0800a12]  居吾將告汝  (居若坐也。謂我將此理告之與汝也)  。

[0800a13]  吾昔甞學於魚巖老人  (魚巖山名。老人即妙明子。秉授法要師祥公是也)  。

[0800a14]  三年之內執弟子禮  (三年之間。以弟子之禮而侍奉也)  。

[0800a15]  不敢以一言形是義也  (不敢輕用一言問此義也)  。

[0800a16]  而魚巖亦不我之告也  (魚巖亦不告說與我也)  。

[0800a17]  五年之後。服勤驅役無不從適  (至五年後。因服事殷勤驅馳役  使。無有不從順而隨適之者也)  。

[0800a19]  始得魚巖引而並席。徵申此道少有開覺  (於此方得  同席而坐。少申道理。略有開解知覺)  。

[0800a21]  爾後日往而問。而不我吝也  (於後日每去問而不慳吝於我)  。

[0800a22]  而子獨心意迫速。而容易於道者乎  (豈惟以心逼迫急速容  易。而便得此道者乎)  。

[0800a24]  客曰。若義奈何  (若此為之奈何)  。

[0800a25]  妙明曰。止乎止乎  (止者。禁約之辭。重言止者。禁之至也)  。

[0800a26]  夫走之愈數而跡之愈多  (夫音扶。夫者起辭之端。數皆音朔。謂走愈頻數  則踪跡愈多)  。

[0800b02]  言之愈數而道之愈遠  (言頻數則道轉不相涉也)  。

[0800b03]  若不處陰而息跡忘言而體道也  (莊子云。處陰以休影。處靜  以息跡。少言辭則道自近矣)  。

[0800b05]  向吾所說。始之以言言言  (向上吾之言辭。則以言其說道之言也)  。

[0800b06]  次之以言言無言  (次後言辭。則用言無言之道也)  。

[0800b07]  然未及以無言言無言也  (若此則猶未是離無言無說之言也)  。

[0800b08]  客曰。請聞無言  (欲聞無言之道)  。

[0800b09]  妙明示之默  (良久默然。以示無言之道)  。

[0800b10]  有間  (有少頃間)  。

[0800b11]  客曰。此離言之道者乎  (只此可謂離言之道者乎)  。

[0800b12]  妙明不應。遂擊几而歌曰  (以手擊几。而遂歌數曰)  。

[0800b13]  脩竹搖風兮窓戶冷。澄潭浸月兮星斗寒瑤  琴無絃兮曲朗然。高山流水兮不須彈  (若邵子詩云。
月到天心處。風來木面時。一般清意味。料得少人知。自擊几主此歌正顯無言之道)  。

[0800b16]  歌罷。就枕而臥  (於是露盡離言之道。學者於此可玩味而自消息之)  。

[0800b17]  客離席出戶。移時而來曰向之問答。理甚隱  密。不近人情。請置其說  (離其席出於門外。過時而來曰。若向所言問答之
辭。其理甚隱而深密。不近尋常情意。且止其說也)  。

[0800b20]  敢問。佛向雪山修道。不云登僊還言成佛。其  義若何  (謂再敢問。佛於雪山修道。不言成仙猶言成佛。其義何也)  。

[0800b22]  妙明曰。道者導也。導人至於無為即導引之  義也  (謂導引眾生至於無為之義也)  。

[0800b24]  客曰。無為者何也  (無為之理。今何者是)  。

[0800b25]  妙明曰。向不云乎。法身佛者即無為也  (言法身佛  者。即無形無相無作無為之妙理也)  。

[0800b27]  然無為者  (下文形容無為之妙)  。

[0800c01]  牽之無前  (如顏子曰。忽然在後是也)  。

[0800c02]  引之無後  (若瞻之在前也)  。

[0800c03]  舉之無上  (若仰之彌高也)  。

[0800c04]  抑之無下  (如抑按無所下)  。

[0800c05]  迎之無首  (欲迎而觀之。則無端緒)  。

[0800c06]  隨之無後  (隨而視之。則無其末後)  。

[0800c07]  視之無形  (顧視不見其形)  。

[0800c08]  聽之無聲  (審聽不聞其聲)  。

[0800c09]  溷而不淈  (溷昏一聲。淈古沒反。皆濁義。謂濁而不濁)  。

[0800c10]  澄而不清  (澄清也謂清而不清)  。

[0800c11]  不可以言言  (法身之理。舉念則乖。故不可以言說)  。

[0800c12]  不可以識識  (識情一動障蔽真如。故不可以識識)  。

[0800c13]  鴻濛渺漭  (鴻者自然之元氣。以喻法身。濛者濛濛然者喻法身似有而無。渺漭者大水貌。亦喻法身  渺漭而無有涯涘矣)  。

[0800c15]  唯如如智可冥會耳  (唯通達如如之智者。可玄知而密會契耳)  。
