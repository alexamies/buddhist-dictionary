大唐沙門慧苑撰

[0441b06]  經從第十七盡第五十

[0441b07]  逮於無上  (逮唐愛反邇雅曰逮與也與即古之預字今經意謂得預無上菩提果也)  。

[0441b08]  喉吻  (吻無粉反蒼頡篇吻謂脣兩頭邊也)  。

[0441b09]  預流  (珠叢曰凡事相及為預也此中謂於見道第十六心得果捨向及聖眾行流故曰預流也)  。

[0441b10]  阿羅漢  (案梵語中此名含攝多義依大婆沙論第九十四中四義釋一者應供二者煞賊三者不生四者遠
惡依唯識論三義以釋一己永害煩惱賊故二應受世間妙供養故三永不復受分段生故依成實論中一釋謂斷惑盡  故名阿羅漢舊翻之為無生者謂三界惑盡更無三界生故)  。

[0441b13]  羯磨  (此云辦事謂諸法事由茲成辦)  。

[0441b14]  和上  (案五天雅言和上謂之塢波地耶然其彼土流俗謂和上殟社于闐  [跳-兆+(梳-木)]  勒乃云鶻社今此方訛音謂之和上雖
諸方舛異今依正釋言塢波者此云近也地耶者讀也言此尊師為弟子親近習讀之者舊云親教是也)  。

[0441b16]  阿闍梨  (此云軌範範即是師義謂與弟子為法則也)  。

初發心功德品

[0441b18]  寧為多不  (寧乃亭反玉篇曰寧案也漢書集注曰案焉也安焉之言皆是微責之詞耳焉音於言反也)  。

[0441b19]  哥羅分  (正云迦羅此云竪析人身上一毛為百分中之一分也或曰十六分中之一分或議譯為校量分  迦音畺佉反)  。

[0441b21]  優波尼沙陀分  (正云塢波尼煞曇言塢波者近也尼煞曇者少也謂少許相近比類之分也或
曰優波此云近也尼沙陀隨也謂相近比對分也或云極也謂數中之極此中經意無限善根多少俱無比對設少許亦  無限極也)  。

[0441b24]  且置  (廣雅曰置捨也)  。

[0441b25]  須陀洹  (正云窣路陀阿鉢囊言窣路陀者此云入也阿鉢囊者此云流也謂遍斷見惑捨異生性初獲聖性  入聖行流故名入流也舊安洹字莫知其所以也)
。

[0441c02]  斯陀含  (此云一來謂此聖者雖斷欲界修惑六品然為有餘三品未斷令聖者一度來生欲界故名一來也)  。

[0441c03]  阿那含  (此云不還斷欲修九品惑盡從此上生色界更不還來受生欲界故名不還也)  。

[0441c04]  傘  (乘亶反)  。

[0441c05]  纔  (昨來反廣雅曰纔暫也)  。

[0441c06]  難制沮  (沮才與反李琦注漢書制斷也禁也毛詩傳曰沮止也)  。

[0441c07]  不告勞  (孟康注韓詩曰古名吏休假曰告也又案告謂告訴即  [(受-又+ㄆ)*辛]  憚之言故詩曰王事鞅掌不敢告勞是也)  。

[0441c08]  覲謁  (覲渠恡反謁於歇反珠叢曰覲謂就見尊老也杜注左傳曰謁白也謂啟白溫清起居之事也)  。

[0441c09]  鑽仰  (鑽則官反論語曰子見齊縗者冕衣裳者與瞽者見之雖少必作過之必趨顏淵喟然歎曰仰之彌高鑽之彌
堅何晏注云言其不可窮盡也謂道高且堅都無際限再仰益高再鑽益堅也)  。

[0441c11]  躭味  (躭都含反案玉篇字林等嗜色為媅嗜酒為躭耳垂為躭聲類媅字作妉今經本作躭字時俗共行未詳  所出也)  。

[0441c13]  珍座  (賈注國語曰珍寶也)  。

[0441c14]  乍可量  (廣雅暫也)  。

[0441c16]  超諸等列  (韻圃稱等齊也杜注左傳曰列位也鄭注禮記曰列等比也張湛注列子曰禽獸之智有與人  同居而有群行則有列列言位也)  。

[0441c18]  文相連屬  (成文相連對文相屬)  。

[0441c21]  皆無瑕玷  (瑕行加反玷丁念反廣雅曰瑕裂也毛詩傳曰玷缺也凡物之有過者皆猶玉之瑕玷古來以  為通語)  。

十行品

[0442a05]  無屈撓行  (撓女教及如紹二反杜注左傳曰撓曲也漢書集注曰撓弱也此中文意明精進波羅蜜勇捍
策勤無退屈無怯弱也撓字正應從木經本有從扌者音呼高反撓擾之字非此所用也)  。

[0442a07]  靡所資贍  (考工記曰資取也小雅曰贍足也言觸事乏少無取足處也又郭象注莊子曰資給濟也聲類  曰贍助也言孤煢無人濟助也)  。

[0442a09]  補伽羅  (正云補特伽羅此曰數取趣謂造集不息數數取苦果也)  。

[0442a10]  摩納婆  (或云摩那婆此曰年少者或謂儒童也)  。

[0442a11]  姝麗  (姝昌逾反說文曰姝色美也小雅曰麗著也謂顏色鮮著)  。

[0442a12]  珍玩  (玩五段反廣雅曰珍重也賈注國語珍美也書曰玩人喪德玩物喪志孔安注曰以人為戲弄則喪德玩物則
喪志也今此謂所愛重戲弄之具也字又作翫妧兩體或戲弄人為妧狎習為翫貪愛為玩)  。

[0442a14]  若或從事  (尒雅曰或有也河上公注老子曰從為也)  。

[0442a15]  頗能  (頗普我普俄二反廣雅曰頗少也)  。

[0442a16]  阿鼻地獄  (阿鼻此云無間也)  。

[0442a17]  慶幸  (何耿反劉兆注公羊傳曰幸遇也韻圃稱幸賴也)  。

[0442a19]  無所顧戀  (廣雅曰顧眷也)  。

[0442a23]  匪懈  (匪方尾反孝經曰夙夜匪懈以事一人鄭注云匪非也懈墮也案非猶不也無也)  。

[0442b01]  無所適莫  (適丁歷反蜀志諸葛亮曰事以覆踈易奪為益無莫為平人情苦親親而踈踈故適莫之
道廢人皆樂人從已不樂已從人故易奪之義廢也論語云子曰君子之於天下無適無莫也漢書集注曰適主也尒雅  曰莫定也謂普於一切無偏主親無偏定踈)  。

[0442b08]  該  (廣雅曰該包也)  。

[0442c05]  我等欽風  (孔安國注書曰欽敬也風教也鄭玄注周禮曰風者聖賢治道之遺化也今謂敬用其教令  也)  。

[0442c07]  都不可得  (漢書拾遺曰都  [撼-咸+(句-口+夕)]  也)  。

[0442c08]  我身薄祜  (祜胡古反鄭箋詩曰祜福也字冝從示邊作古經本有示邊作右者音尤救反孔子述易曰祐

[0442c10]  諸根殘缺  (缺傾雪反蒼頡篇曰殘傷也)  。

[0442c11]  不淨微形  (毛詩傳曰微瘍也瘍音移益反郭注邇雅曰瘍瘡也言身之不淨如瘡也又王注書曰微賤也  言此不淨之身非可貴重也瘍微從彳賤微也)  。

經卷第二十二　昇兜率天宮品

[0442c21]  綺煥  (煥呼換反張載注靈光殿賦云綺文也何晏注論語曰煥明也言其文彩分明也)  。

[0442c22]  頻婆帳  (頻婆此云身影質謂此帳上莊嚴具中能現一切外質之影也或曰頻婆鮮赤菓名此帳似之故以  名也)  。

[0443a02]  香氣發越  (漢書集注曰發越謂香氣射散也言香氣纔發四散著人如箭射速中也)  。

[0443a06]  樓閣延袤  (袤莫搆反尒雅曰延長也切韻稱袤廣也史記曰蒙恬築長城延袤萬餘里是也)  。

[0443a09]  因於撫擊  (撫孚武反字正宜作拊廣雅曰相也釋名曰拊敷也敷以手指之指搏也經本作撫字者此乃  撫育安撫之字者也)  。

[0443a11]  克諧眾樂  (尒雅曰克能也孔安注書曰諧和也言能以眾多音樂和合成諸歌曲也)  。

[0443a12]  稽首作禮  (周禮太祝辨九拜之儀一曰稽首再拜頭至地也二曰頓首再拜頭叩地也三曰空首再拜頭  至手也四曰振首戰動拜也五曰吉祥拜
[齋-小+衣]  縗不拜以下拜也六曰凶拜稽顙而後拜謂三年服者七曰奇拜謂一拜八  曰褒拜謂報拜也九曰肅拜謂但俯而下手也)  。

[0443a15]  不瞬  (瞬舒閏反說文曰瞬謂目開閇數搖也字正體作瞚今並隨俗作瞬也)  。

[0443a16]  阿迦尼吒天  (具云阿迦尼瑟吒言阿迦者色也尼瑟吒究竟也言其色界十八天中此最終極也又云
阿無也迦尼瑟吒小也謂色界十八天中最下一天唯小無大餘十六天上下互望亦大亦小此之一天唯大無小故以  名也)  。

[0443a22]  良沃田  (沃烏鵠反漢書集注曰沃溉灌也言其土地有溉灌之利也今謂溉灌之田復加肥善故曰良沃田)  。

十迴向品

[0443a24]  瞋佷  (佷何懇反杜注左傳曰佷戾也說文曰佷不聽從也案玉篇佷字在彳部今多從立人葢是時俗共行之)  。

[0443b01]  崇巖邃谷  (  [遽-虍+穴]  辛醉反鄭注考工記曰崇高也說文曰  [遽-虍+穴]  深也  [遽-虍+穴]  字從穴遂聲也)  。

[0443b02]  入苦籠檻  (檻胡黤反籠字正冝作櫳三蒼曰櫳所以宬禽獸闌檻也說文曰櫳牢也郭璞注山海經曰檻
闌也謂穿地為坑上安櫺子以閉禽獸也今經意謂三界苦如彼櫳檻囚繫眾生或謂三途劇苦名苦櫳檻也)  。

[0443b04]  眾苦大壑  (壑訶各反兼名苑曰東海有大燋石一名沃燋方圓三萬里水沃之則消盡過此有大壑一名
尾閭深莫測其涯海水常澍不知其所之也今經意謂生死海中三苦八苦無有涯底如彼大壑故借喻言耳也)  。

[0443b06]  五欲所致  (顏注漢書曰致謂引而至之也)  。

[0443b07]  志獨無侶  (鄭注禮記曰志意也侶伴也謂廣心獨濟不待伴也)  。

[0443b08]  顧復一切眾生  (復扶福反詩云父兮生我母兮掬我拊我畜我長我育我顧我復我出入腹我
鄭玄牋曰顧視也復反也言去子雖近猶步步反視也又牋詩云是顧是復者顧念而重復之也今經意菩薩於眾中亦  然矣)  。

[0446b19]  一俱盧舍  (依毘曇中一尺五寸為一肘。四肘為一弓。三百弓為一里。四里為一俱盧舍。準計一里三百六
十步。則一俱盧舍有一千四百四十步也。依俱舍論第十二云。分析諸色至一極微故一極微為色邊際。七極微為一微
量積微至七為一金塵。積七金塵為水塵量水塵積七為一兔毛塵。積七兔毛塵為羊毛塵量積羊毛塵七為一牛毛塵。  七牛毛塵為隟遊塵量隟塵七為一蟣量。積七蟣量為一
[蝨-十+虫]  。七  [蝨-十+虫]  為穬麥。七麥為指節。二十四指橫布為肘竪。四肘為弓  竪積五百弓為一俱盧舍。即是從肘至阿練若中間道量)  。

[0446b24]  半由旬量  (由旬是梵言具云踰繕那。此曰和合。謂以多俱盧舍和合成故。案佛本行集第十二云。七微塵  成一  [穴/匆/心]  牖塵。七
[穴/匆/心]  成一兔毛頭塵。七兔毛塵成一羊毛頭塵。七羊毛塵成一牛毛塵。七牛毛塵成一蟣。七蟣成一虱。七虱
成一芥子。七芥子成一大麥。七大麥成一指節。七指節成尺。二尺成一肘。四肘成一弓。五弓成杖。二十杖名一息。八十息
名一俱盧舍。八俱盧舍成一由旬。準此方尺量二里餘八十步當一俱盧舍。計一由旬合有一十七里餘二百八步或百  一由旬。如此方一驛地也)  。

[0448a11]  其掌安平  (掌謂舌面平而且安)  。

[0448a12]  玩味不忘  (玩字正宜作忨注左傳曰忨貪也經木作玩者誤也案孔安國注尚書云玩戲貪弄忨即是愛  樂之意也)  。

[0448a14]  上咢  (俄各反正體作咢又或作齶也)  。

[0448a15]  右輔下牙  (左傳宮之奇曰虞  [虢-寸+(百-日+ㄎ)]  之表也猶輔車相依脣亡齒寒杜注曰輔頰也車牙車也又注曰輔頰車  骨也)  。

[0448a17]  彌盧藏雲  (彌盧此云高以在佛止牙故也)  。

[0448a18]  紺蒲成就  (紺蒲正云劒蒲此乃西域菓名其色紅赤腹圓三約橫文而佛頸成就彼相故云也今時俗謂  頸圓有約為嬰節者是也)  。

[0448a20]  彌布十方  (顏注漢書曰彌滿也)  。

[0448a21]  左臋  (臋徒論反聲類尻也尻音苦勞反也)  。

[0448a22]  其  [月*坒]  與膊  (  [月*坒]  字正冝作髀古文作蹕今  [月*坒]  未詳所出膞字宜作腨今經本作膞者謬也)  。

[0448a23]  伊尼延鹿王腨  (伊尼者鹿名也其毛色多黑腨形  [月*庸]  纖長短得所其鹿王最勝故取為喻腨字  又作  [跳-兆+專]  也)  。

[0448b01]  舉足將步  (將欲也步行也王逸注楚辭曰步徐行)  。

[0448b02]  足跟  (跟各痕反)  。

[0448b03]  因陀羅尼羅  (因陀羅此云帝也尼羅此云青也)  。

如來隨好光明功德品第三十五

[0448b05]  隨好  (好呼告反令眾生愛樂故又音呼暠反資嚴大相蓋殊勝故)  。

[0448b06]  摩耶夫人  (摩耶此云幻也由此菩薩唯依大願智幻法門行故故以此名夫人者梵本云弟脾此翻為女
天案鄭注禮云諸侯之妃曰夫人玉篇曰呼婦人為夫人者亦所以崇敬之稱也又夫者男子美稱婦因夫以成人故名  夫人也西域呼王妃為弟脾呼男夫為弟婆也)  。

[0448b09]  懺悔  (懺謂懺摩此云謂忍謂請前人忍受我悔罪)  。

經卷第四十九　普賢行品第三十六

[0448b11]  樂近凡庸  (廣雅曰凡輕也漢書集注凡庸微小也言其輕薄寒微眇小之人耳也)  。

[0448b12]  習童蒙法  (易稱蒙卦者謂來求我非我求蒙蒙者  [(懞-一)-卄+((並-(前-刖))-一)]
也韓康注云蒙昧幼小之蒙又玉篇曰童幼迷昏也  廣雅曰童癡也鄭玄注周禮蒙冒也毛詩傳曰蒙覆也言童幼之心愚昧所為惽謬如以物覆蔽也)  。

[0448b14]  口如啞羊障  (啞於雅反不能宣說大乘妙義名為啞羊障大智度論第三云亞羊僧者謂雖不破戒鈍
根無慧不別好醜不知輕重不知有罪若有僧事二人共諍不能斷決默然無言譬如白羊乃至人殺不能作聲是名啞  羊僧)  。

[0448b17]  洎乎法界  (正冝作暨暨至也音渠祕反珠叢曰暨謂及預也杜注左傳曰暨至也經作洎者誤也)  。

[0448b18]  充洽  (洽咸夾反小雅曰充  [佫-口+用]  也玉篇曰洽灑也濡沽潤)  。

[0448b19]  工幻師  (韻圃稱工巧也謂巧能於幻術也)  。

經卷第五十　如來出現品第三十七之一

[0448b21]  偏袒  (  袒  唐亶反露也字從示又音宅莧反非此所須)  。

[0448b22]  右跽  (跽其几反跪也)  。

[0448b23]  如我惟忖  (忖忩本反顏注漢書曰惟思也珠叢曰忖測度也)  。

[0448b24]  瞻仰如來仁及我  (有仁德者号之為仁故謚法曰貴親親曰仁殺身成人曰仁度功而行曰  仁周禮云天德曰仁禮記曰上下相親曰仁論語云剋  己
復禮曰仁鄭玄注禮記曰施恩曰仁白虎通曰仁者是忍好生  愛人也)  。

[0448c03]  佉陀羅山  (爾雅曰軻梨羅此即木名謂此方苦棟木也由彼山中多有此木故立其名)  。

[0448c04]  尼民陀羅山  (此翻為持邊以彼山是七重金山中最外邊故然即院統護持餘內六山故名持邊)  。

[0448c05]  目真隣陀山  (目真此云解脫即是龍名也隣陀此云處也謂此山中是解脫住處)  。

[0448c06]  水族眾生  (孔安注書曰族類也謂水畜之流類)  。

[0448c07]  大雨名洪霔  (霔之庶反洪大也併急寫水日霔)  。

[0448c08]  變溼令燥  (燥蘇早反燥乾也變字從戀者作又有從反者不是字也溼字有作濕者誤也)  。

[0448c09]  廓徹虛空  (廓苦郭反邇雅曰廓大也今謂寬大遍虛空量也)  。

[0448c10]  普照無私  (玉篇曰事不公為私王逸注楚辭曰竊愛為私也郭象注莊子曰世所謂無私者釋已而愛人  也言無已情偏有所為也私字么上加撇者非)  。
