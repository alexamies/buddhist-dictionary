Tibetan Language Resources

This page describes how to work with Tibetan text, especially working between Chinese and Tibetan. There is more information on some subjects in Buddhism Tibetan than Chinese, some subjects are specifically Tibetan, and some people are more famililar with Tibetan than Chinese. This page is written with those needs in mind.

Mahāvyutpatti Sanskrit-Tibetan-Chinese Buddhist Dictionary Chrome Extension

The Mahāvyutpatti Sanskrit-Tibetan-Chinese Buddhist Dictionary Chrome Extension adds a context menu when text on a web page is selected so that term can be used to lookup and entry in the dictionary. You can do multi-term lookup for longer Chinese phrases, and multi-language lookup with Sanskrit and Tibetan. See this demo on Youtube for details or the Support page on how to use it.

Tibetan Text in NTI Reader Multilingual Entries

The NTI Reader is primarily a Chinese to English resource. However, Tibetan equivalents are also provided for many terms. The goal of these is to aid in discovery of terms and navigation between different resources. The Tibetan equivalents are given in Wylie transliteration. To convert between Wylie and Tibetan script or ‘Unicode’ use a tool like the converter from the Tibetan and Himalayan Library.
